# Credit Card Fraud Detection using PySpark

This project implements a complete machine learning workflow for detecting fraudulent credit card transactions using **PySpark**.  
It is based on the well-known Kaggle dataset *‚ÄúCredit Card Fraud Detection‚Äù*, which contains highly imbalanced data representing real-world fraud detection challenges.

The solution is developed using two separate Jupyter notebooks:

- **01_eda_and_preprocessing.ipynb** ‚Äì exploratory data analysis and data cleaning  
- **02_modeling_and_evaluation.ipynb** ‚Äì model training, prediction, and performance evaluation  

Both notebooks rely on **Spark MLlib** for scalable machine learning.

## üöÄ Project Overview

The goal of this project is to:

- Load and explore the credit card dataset
- Perform data cleaning and type casting
- Conduct exploratory data analysis (EDA)
- Assemble and scale features for MLlib models
- Train two machine learning models:
  - Logistic Regression  
  - Random Forest Classifier
- Evaluate model performance using:
  - ROC‚ÄìAUC  
  - F1-score  
  - Accuracy  
- Discuss the limitations caused by the high class imbalance

This project demonstrates how PySpark can be used effectively on medium-scale datasets while applying big-data machine learning pipelines.
